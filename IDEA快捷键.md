## IDEA快捷键

<!-- more -->

| 快捷键 | 描述 |
| :----:  |  :--:  |
| Alt + J | 选中多个匹配项 |
| Alt+Shift+鼠标点击 | 出现多个光标 |
| Ctrll+Shift+up/down | 向上/下移动代码行 |
| Alt+up/down | 向上/下跳转函数 |
| Ctrl+D | 向下复制一行 |
| Ctrl+Alt+L | 格式化 |
| Ctrl+X | 删除当前行 |
| Shift+Enter | 向下插入新行 |
| Shift+F6 | 变量重命名 |
| Shift+Del | 剪切当前行 |
